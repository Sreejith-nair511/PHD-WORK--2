# data/raw/ — Dataset Storage

This folder holds all raw, unprocessed datasets.
It is **git-ignored** — nothing here gets committed to GitHub.

Place each dataset in its own subfolder:

```
data/raw/
├── daic_woz/     ← DAIC-WOZ dataset (Speech + Text + Face)
├── modma/        ← MODMA dataset (Speech + EEG)
└── pdch/         ← PDCH dataset (Speech + Text + Face + EEG)
```

See the main README.md for exact folder structures and download links.
